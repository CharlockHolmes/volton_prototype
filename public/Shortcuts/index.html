Planner Shortcuts Description
CTRL+Z Undo
CTRL+S Save planner configurations
WASD keys Move component
Arrow keys Move component
E - Q Change component width
R - F Change component height
C Center component
M Make a copy of the component at +180 degrees
L Lock component into position so that you cant move it with your mouse
T Translate component to cursor position
Mouse Wheel Move the planner scrollbar


Interface Shortcuts Description
O Reset the camera position
H Make tutorial transparent
F1 Go to previous tutorial step
F2 Go to next tutorial step
